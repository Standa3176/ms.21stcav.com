- `CompetitorFtpSourceResource` Filament admin Resource with 'Test connection' + 'Pull now' actions (D-08, D-09)
- `AlertRecipientResource` form/table extension for the new toggle
- Schedule entry every 15 min (D-07 — runs BEFORE the every-5-min watcher slot so the next watcher tick picks up the file)
- `shield:safe-regenerate` + RolePermissionSeeder LIKE pattern `%_competitor_ftp_source` (admin only — pricing_manager/sales/read_only get NO perms; this is an admin tool with credentials)
- 6 Pest tests + PolicyTemplateIntegrityTest floor bump 29→30

Net stack delta: ZERO composer changes (`league/flysystem-ftp` + `league/flysystem-sftp-v3` are already transitively pulled by Laravel 12). Phase 11.1 lives INSIDE the existing `Competitor` Deptrac layer (CONTEXT.md Claude Discretion) — NO `depfile.yaml` / `deptrac.yaml` edits required.

Purpose: Without this plan, ops still depends on n8n/manual file drops to populate `storage/app/competitors/incoming/`. With this plan, an admin can manage FTP/SFTP sources via Filament, "Test connection" before going live, and let the scheduler pull every 15 min — closing the COMP-01..04 fetch-step gap.

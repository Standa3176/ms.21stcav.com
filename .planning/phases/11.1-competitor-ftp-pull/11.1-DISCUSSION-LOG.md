## Pull flow (auto)

`competitor:ftp-pull` artisan command, dry-run default + `--live` opt-in (Phase 2 D-04 pattern), `--source={id}` scope flag. Atomic write via tempnam→rename. Filename normalisation to Phase 5 `{slug}_{YYYY-MM-DD}.csv` if remote name doesn't match. Updates source row state + audits via Auditor. Schedules every 15 min via `routes/console.php` with `withoutOverlapping(20)`.

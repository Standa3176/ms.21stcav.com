## Hand-off

Phase 11.1 ships the missing fetch step. Operators can now:
1. Run `php artisan db:seed --class=RolePermissionSeeder` to attach the new perms.
2. Visit `/admin/competitor-ftp-sources/create` to add a competitor FTP source (fill host/port/credentials/regex/cron/verify_ssl).
3. Click "Test connection" to verify credentials before flipping `is_active=true`.
4. Click "Pull now" for an immediate one-shot pull, or wait for the next 15-min schedule slot.
5. Files land in `storage/app/competitors/incoming/` matching Phase 5's `{slug}_{YYYY-MM-DD}.csv` convention.
6. Phase 5's `competitor:watch` (every 5 min) picks them up via the >30s mtime gate and runs the unchanged parse → DB → margin analyser pipeline.

When a source fails 3 consecutive pulls, the source is auto-disabled and AlertRecipient rows with `receives_competitor_ftp_alerts=true` receive a `CompetitorFtpPullFailedNotification` mail.

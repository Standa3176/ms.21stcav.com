From app/Domain/Agents/Services/GuardrailEngine.php (Phase 8):
- `runPreFlight(RunsAsAgent $agent, array $input, TrustTier $tier): array`
- `runPostFlight(RunsAsAgent $agent, ClaudeResponse $response, TrustTier $tier): ClaudeResponse`
Iterates the agent's `guardrails()` array. For each one:
  - If `$g->isPreFlight()` AND `$g->shouldRun($tier)` → calls `$g->pre($input)` (may modify input)
  - If `$g->isPostFlight()` AND `$g->shouldRun($tier)` → calls `$g->post($response)` (may modify response OR throw)

Guardrail interface (verify exact name by reading existing Pricing guardrails):
```php
interface Guardrail
{
    public function isPreFlight(): bool;
    public function isPostFlight(): bool;
    public function shouldRun(TrustTier $tier): bool;
    public function pre(array $input): array;       // pre-flight only — guardrail can leave as no-op
    public function post(ClaudeResponse $response): ClaudeResponse;  // post-flight only — may throw GuardrailViolationException
}
```

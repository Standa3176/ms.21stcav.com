<objective>
In the Pricing Operations bucket popup (the tile modal table: SKU / Name / Our cost (ex) / Lowest comp (ex) / Margin), show WHERE each price comes from:
- the **supplier name** behind "Our cost (ex)" (= the cheapest current SupplierOfferSnapshot for that product, matching how buy_price is chosen),
- the **competitor name** behind "Lowest comp (ex)" (= the winning competitor in the lowest-across-competitors selection).

Purpose: operators viewing a below-cost / at-floor row need to know which supplier we're costing at and which competitor is undercutting, without leaving the popup or running `supplier:explain-cost`.

Output: `CompetitorPositionScanner::compute()` returns two new per-row fields (`supplier_name`, `competitor_name`, both `string|null`), resolved with BATCHED lookups (no N+1). The bucket blade renders them as compact muted sub-lines. Inline page panels + CSV export are untouched.

Scope guardrails:
- BUCKET POPUP ONLY for the visible change. Do NOT change `pricing-operations.blade.php` inline panels or `PricingOpsReport::csv()`.
- Display only — no margin/pricing-logic change, no schema change, no new endpoints.
</objective>

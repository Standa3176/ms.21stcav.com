   $generatedAt.

   on success: Cache::forget('taxonomy.brands'); remove $brand from the cached summary + from $this->brands;
   Notification::make()->success()->title("Brand '{$brand}' created on Woo")->send(). On \Throwable →
   Notification danger with the message. (Catch woocommerce_term_exists gracefully → treat as success/info.)
 - The Blade view renders a table of $brands (brand | products it would unlock | sample SKUs) each with a
   wire:click="createBrand('...')" button (button hidden for non-writers via @if(auth()->user()?->hasAnyRole(...))),
   the generated_at line, and an empty-state when $brands is empty.

WooClient: post(string $endpoint, array $payload): array — goes through writeOrShadow (write_enabled=true
in prod). products/brands is the WC-native brands taxonomy TaxonomyResolver::allBrands() reads, so a new
term there makes the product creatable. WooClient is subclassable for tests (anon subclass recording post()).

SuggestionResource (app/Domain/Suggestions/Filament/Resources/SuggestionResource.php) filters() array
(~line 235-315) — ADD three filters after the existing on_supplier_db one, all rendered AboveContent:
 - Competitor: SelectFilter::make('competitor')->options(fn()=>\App\Domain\Competitor\Models\Competitor::
   orderBy('name')->pluck('name','name')->all())->query(fn(Builder $q,array $d)=> ($v=$d['value']??null)
   ? $q->whereRaw("JSON_SEARCH(evidence, 'one', ?, null, '$.competitor_sightings[*].name') IS NOT NULL",[$v]) : $q).
 - Brand: SelectFilter::make('brand')->options(fn()=> /* distinct tagged brands */ DB::table('suggestions')
   ->where('kind','new_product_opportunity')->whereNotNull(DB::raw("JSON_UNQUOTE(JSON_EXTRACT(evidence,'$.brand'))"))
   ->distinct()->pluck(DB::raw("JSON_UNQUOTE(JSON_EXTRACT(evidence,'$.brand'))"))->filter()->sort()->mapWithKeys(fn($b)=>[$b=>$b])->all())
   ->searchable()->query(fn(Builder $q,array $d)=> ($v=$d['value']??null)
   ? $q->whereRaw("JSON_UNQUOTE(JSON_EXTRACT(evidence,'$.brand')) = ?",[$v]) : $q).
 - Brand on Woo: TernaryFilter::make('brand_on_woo')->label('Brand on Woo')->placeholder('All')
   ->trueLabel('Ready (brand on Woo)')->falseLabel('Needs brand added')
   ->queries(true: fn($q)=>$q->whereRaw("JSON_EXTRACT(evidence,'$.brand_on_woo') = true"),
     false: fn($q)=>$q->whereRaw("JSON_EXTRACT(evidence,'$.brand_on_woo') = false"), blank: fn($q)=>$q).

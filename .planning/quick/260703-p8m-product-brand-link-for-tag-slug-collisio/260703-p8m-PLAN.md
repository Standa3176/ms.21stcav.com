<objective>
Yealink, Cisco, Logitech, Lenovo, Samsung have no clickable storefront Brand link because a `product_tag`
already owns their clean slug (the pipeline pushes brand-as-tag), so WP REST hard-rejects a clean-slug
`product_brand` create (term_exists), and the resolver's safe `skip-creation` default (post 2026-06-13
duplicate incident) then refuses. Operator decision (2026-07-03): create suffixed brand terms
(/brand/yealink-brand/, name still "Yealink"). Add a `suffix-on-tag-collision` strategy (safe because the
clean slug is provably blocked) as the new default, and a `--brand` filter on resync-to-woo so all of a
brand's products can be relinked in one run. Cosmetic (Brand link only) — everything else is already live.
</objective>

```php
// 260703-p8m — suffix-on-tag-collision: a real product_tag holds the clean

// can NEVER be created for this name → creating {slug}-brand here cannot spawn

    $id = $this->tryCreate($brandName, $primarySlug.'-brand');

        Log::info('product_brand.suffix_on_tag_collision', [
            'brand' => $brandName,
            'colliding_tag_id' => $collision['id'],
            'created_slug' => $primarySlug.'-brand',
        ]);
        return $id;
```

```php
$skus = $this->parseSkus($this->option('skus'));            // existing parse (keep)
$brandNames = array_filter(array_map('trim', explode(',', (string) $this->option('brand'))));
if ($brandNames !== []) {
    $taxonomy = app(TaxonomyResolver::class);
    $wantLower = array_map('mb_strtolower', $brandNames);
    $brandIds = [];
    foreach ($taxonomy->allBrands() as $b) {
        if (in_array(mb_strtolower(trim((string) ($b['name'] ?? ''))), $wantLower, true)) {
            $brandIds[] = (int) $b['id'];
        }
    }
    if ($brandIds !== []) {
        $brandSkus = Product::whereIn('brand_id', $brandIds)
            ->whereNotNull('woo_product_id')
            ->pluck('sku')->filter()->map(fn ($s) => (string) $s)->all();
        $skus = array_values(array_unique(array_merge($skus, $brandSkus)));
    }
}
if ($skus === []) { $this->error('Provide --skus or --brand.'); return SymfonyCommand::FAILURE; }
```

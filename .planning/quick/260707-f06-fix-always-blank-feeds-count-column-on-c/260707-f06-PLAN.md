<objective>
The Feeds column on Settings > Competitor Feeds (Competitors list) is ALWAYS blank. Root cause:
`TextColumn::make('feeds_count')->counts('ftpFeeds')` — `->counts('ftpFeeds')` adds `withCount('ftpFeeds')`
which yields the attribute `ftp_feeds_count` (Laravel snake-cases the relation), but the column is keyed
`feeds_count`, so it reads a non-existent attribute → blank. (Probe confirmed the real attribute is
`ftp_feeds_count`=1.) Rename the column key to `ftp_feeds_count` so it matches. One-line fix + a guard test.
</objective>

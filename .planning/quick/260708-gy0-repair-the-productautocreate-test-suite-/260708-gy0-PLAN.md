```php
/** Queued-listener queue selector — listeners have no onQueue(); viaQueue() is the sanctioned hook

}
```

=== (3) Suggestion — booted() creating defaults (prod audit-write hardening) ===
Add to the model (imports: `use Illuminate\Support\Facades\Context;` `use Illuminate\Support\Str;`):
```php
protected static function booted(): void
{
    static::creating(function (self $s): void {
        // payload is NOT NULL in the schema; many kinds carry data only in `evidence`.
        if ($s->payload === null) {
            $s->payload = [];
        }
        // correlation_id is NOT NULL; failure-audit hooks may run outside a correlated context.
        if ($s->correlation_id === null || $s->correlation_id === '') {
            $s->correlation_id = Context::get('correlation_id') ?? (string) Str::uuid();
        }
    });
}
```
Only fills when MISSING — explicit values pass through unchanged. This fixes the failed()-hook writes AND the
test-arrange inserts that omit these columns.

2. lookupGtinByMpn becomes the orchestrator (keep the signature + empty guard):
```php
public function lookupGtinByMpn(?string $brand, ?string $mpn): ?string
{
    $mpn = $mpn !== null ? trim($mpn) : '';
    if ($mpn === '') {
        return null;
    }

    $ean = $this->queryBarcode($mpn);

    // Region-localized SKUs (HP '#ABU' UK, '#ABB', '#AC3', '#UUZ', …) — EAN databases
    // list the GTIN under the BASE part number. Retry ONCE with the suffix stripped.
    if ($ean === null && str_contains($mpn, '#')) {
        $base = trim((string) strstr($mpn, '#', true)); // part BEFORE the first '#'
        if ($base !== '' && $base !== $mpn) {
            $ean = $this->queryBarcode($base);
        }
    }

    return $ean;
}
```
Do NOT strip '/' or spaces — only '#'. Keep `$brand` in the signature (unused, as today). Keep all logging
inside queryBarcode so both attempts are recorded in integration_events.
